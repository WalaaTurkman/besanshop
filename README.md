# 🛍️ BesanShop

BesanShop is a simple e-commerce website focused on selling bags and mugs. The project was built using HTML, CSS, and JavaScript to provide a clean and user-friendly shopping experience.

---

## 📌 About the Project

This project is a frontend web application that simulates an online store. It allows users to view products, and interact with a simple shopping interface.

The main goal of the project is to practice building responsive layouts and interactive UI using vanilla web technologies.

---

## ✨ Features

- 👜 Display products (bags & mugs)
- 🎨 Clean and responsive UI design
- 🧠 Simple JavaScript interactivity
- 📱 Mobile-friendly layout

---

## 🧰 Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 🎯 Purpose of the Project

- Practice frontend development fundamentals  
- Improve UI/UX design skills  
- Build a small e-commerce prototype  

---

## 🚀 Future Improvements

- Add shopping cart functionality  
- Add product filtering and search  
- Connect to backend database  
- Add user login system
